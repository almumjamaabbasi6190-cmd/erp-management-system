# ERP Management System (MERN Stack)

This project is an ERP Management System built using the MERN stack (MongoDB, Express, React, Node.js).

## Features
- User Authentication (Admin Login)
- Inventory Management
- Orders Management
- Sales Management
- Suppliers Management
- Payments (Backend)
- Role-based access (Admin)

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## How to Run

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npm start
